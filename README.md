🚀 JanSahay AI — Citizen Assistance Platform

🇮🇳 Your Smart Guide to Government Schemes in India

JanSahay AI is an AI-powered web application designed to simplify access to government schemes for Indian citizens. It enables users to discover, understand, and apply for welfare schemes through an intuitive interface, multilingual support, and smart assistance features.

✨ Key Features
💬 AI Chat Assistant
Provides instant responses about government schemes
Simple and user-friendly interaction

🌐 Multi-Language Support
Supports multiple Indian languages
Improves accessibility for diverse users

🔍 Scheme Finder
Suggests schemes based on user inputs

Displays user-related information

🔑 Setup Instructions
1️⃣ Clone the Repository
git clone https://github.com/hansagusaiwal/Jansahay-AI.git
cd Jansahay-AI
2️⃣ Run the Project
Option 1: Run Frontend Only
Open index.html in your browser
Option 2: Run with Server
node server.js

🔐 Firebase Configuration
To enable authentication:
Create a project in Firebase Console
Enable Authentication:
Email/Password
Google Sign-In
Add your Firebase config in login.html
